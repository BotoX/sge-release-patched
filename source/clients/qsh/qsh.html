QSH - queue shell

The qsh client command submits an interactive job that will start a terminal window (default: xterm) on the submitters screen.

The users login shell is started as command in the xterm window.

For an overview of the qsh options see the output of qsh -help and the qsh manpage qsh(1).

Configuration

Which binary to start as terminal window can be configured in the cluster configuration with the parameter "xterm" - see also the manpage sge_conf(5).
