Select Hosts

This screen is where you select what hosts and what components you want to install. You may use component selection checkbox (not display in express mode) to perform more customized host selection.
By default the qmaster host is added (based on Qmaster host value from the main configuration screen).

We can choose hosts in two ways:

  • by a hostname or host name pattern

  • by an IP address or IP address pattern

Note

The patterns do not support regular expression! Supported are lists and numeric ranges.
See examples below.

Examples:

Description

Input

Resolved as

Hostname

grid00

grid00

IP address

192.168.0.1

192.168.0.1

List of hosts

grid00 grid01 grid05

grid00 grid01 grid05

List of IP addresses

192.168.0.1 192.168.0.2 192.168.0.5

192.168.0.1 192.168.0.2 192.168.0.5

Host ranges

grid[00-10]

grid00, grid01, ..., grid10

Range of IP addresses

192.168-169.0.50-60

192.168.0.50 ... 192.168.0.60, 192.169.0.50 ... 192.169.0.60
