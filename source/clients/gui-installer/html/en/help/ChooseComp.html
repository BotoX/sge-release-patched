Choose component

You may choose what componets are you going to install on this screen.

Possible chooses are:

  • Qmaster host

Qmaster is the main component in Grid Engine software. If you want to install Grid Engine you must install exactly 1 qmaster component per Grid Engine installation.

  • Shadow host(s)

Shadow host provide a high availabality feature to the cluster. In case that the qmaster fails (crash, network issue), one of the shadow hosts will take over the qmaster responsibility

  • Execution host(s)

Hosts that that execute the tasks (jobs).

  • Berkeley db host

Implies a Berkeley db host spooling option. The Grid Engine that spools data to a remote server. Not a recommended default option!
