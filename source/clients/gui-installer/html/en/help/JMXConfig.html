JMX Configuration

  • JMX port - port number to be used by JVM thread

  • Enable SSL server authentication - once enabled SSL certificate configuration will be presented later, the server certificate will be used for authentication and encryption

  • Enable SSL client authentication - client authentication will be used

  • Path to the keystore - path to Java^TM^ keystore file to be created

  • Keystore password - keystore password, default is changeit

  • Retype password - password to retype, default is changeit

  • JVM library path - path to the JVM library on the qmaster host

  • Additional JVM args - additional arguments to be used when starting the JVM in qmaster, default -Xmx256m
